# Reserva de Pistas

Aplicación web estática para la reserva de pistas deportivas.

## Tecnologías
- HTML, CSS, JavaScript
- Firebase Firestore (backend delegado)
- Netlify / GitHub Pages

## Funcionalidades
- Crear reservas
- Listado en tiempo real
- Validación de solapamientos
- Eliminación de reservas
